Crea una plataforma educativa llamada EduAI Companion que tenga:

1. SISTEMA DE AUTENTICACIÓN:
- Página principal con dos opciones:
  * Botón "Obtener Token" para nuevos usuarios (pide email)
  * Campo para ingresar token (usuarios existentes)
- El sistema debe generar y guardar un token único vinculado al perfil
- Después de verificar el token:
  * Usuarios nuevos → ir a cuestionario
  * Usuarios existentes → ir a dashboard

2. CUESTIONARIO INICIAL:
Crear una serie de 6 preguntas secuenciales:

Pregunta 1 - Horario de Estudio:
- Pregunta: "¿En qué momento del día te sientes más productivo para estudiar?"
- Tipo: Opción única
- Opciones: "Mañana", "Tarde", "Noche"

Pregunta 2 - Concentración:
- Pregunta: "¿Cuánto tiempo puedes mantener la concentración en una tarea?"
- Tipo: Opción única
- Opciones: "15-30 minutos", "30-60 minutos", "Más de 60 minutos"

Pregunta 3 - Estilo de Aprendizaje:
- Pregunta: "¿Cómo prefieres aprender nuevo material?"
- Tipo: Opción única
- Opciones: "Viendo videos o imágenes", "Escuchando explicaciones", "Practicando y experimentando"

Pregunta 4 - Rendimiento:
- Pregunta: "¿Cuál ha sido tu rendimiento académico promedio?"
- Tipo: Número
- Escala: 0-10

Pregunta 5 - Objetivos:
- Pregunta: "¿Cuáles son tus objetivos de aprendizaje principales?"
- Tipo: Texto libre

Pregunta 6 - Intereses:
- Pregunta: "¿Qué temas o áreas te interesan más?"
- Tipo: Selección múltiple
- Opciones: "Tecnología", "Ciencias", "Artes", "Humanidades", "Deportes"

3. ALMACENAMIENTO DE DATOS:
Guardar en la base de datos:
- Email del usuario
- Token asociado
- Respuestas al cuestionario
- Estilo de aprendizaje calculado
- Estado de completitud del perfil

4. DASHBOARD DEL USUARIO:
Mostrar:
- Bienvenida personalizada
- Resumen del perfil de aprendizaje
- Recomendaciones personalizadas
- Horario sugerido de estudio
- Progreso del usuario

5. CARACTERÍSTICAS ADICIONALES:
- Barra de progreso en el cuestionario
- Botones Anterior/Siguiente
- Guardar progreso automáticamente
- Mensajes de error claros
- Estados de carga
- Validación de datos
- Rutas protegidas

6. DISEÑO:
- Interfaz limpia y moderna
- Responsive para todos los dispositivos
- Colores consistentes
- Transiciones suaves
- Iconos intuitivos

7. SEGURIDAD:
- Validación de tokens
- Protección de rutas
- Manejo de sesiones
- Validación de emails
- Manejo de errores

Todos los textos deben estar en español y la interfaz debe ser intuitiva y fácil de usar. El sistema debe guardar automáticamente el progreso y permitir retomar el cuestionario donde se quedó el usuario.

Por favor, implementa esta aplicación asegurando una experiencia de usuario fluida y manteniendo la persistencia de datos.
